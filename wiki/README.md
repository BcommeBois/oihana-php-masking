# Oihana PHP Masking — Documentation

[![License](https://img.shields.io/packagist/l/oihana/php-masking.svg?style=flat-square)](../LICENSE)

A lightweight, framework-agnostic PHP 8.4+ toolkit to **anonymize and redact** the fields of your documents — composable standalone maskers, a path-based document engine, strongly-typed enums, zero *magic strings*.
